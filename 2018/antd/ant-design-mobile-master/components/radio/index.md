### 定义／Definition
Radio提供了一个简单的方法来从一个预定义集合中选取单个值。

### 规则 / Rule

- 单选按钮一般出现在列表右侧，只有当列表右侧有内容时，才出现在列表左侧；
- 选择后可以直接触发某个行为。


## API

| name    |         | String |   | 无  |
| checked    |        | String |   | 无  |
| disabled      |         | String | |  无  |
| onChange    | change事件触发的回调函数,参数是event对象 | Function | |   无  |
