title: 大数字
---

数字大于99，会显示 `99+`


````jsx
import { Badge } from 'antd-mobile';

ReactDOM.render(
  <div className="badge-container" style={{ padding: '40px 16px' }}>
    <Badge text={99}>
      <span className="head-example"></span>
    </Badge>
    <Badge text={108}>
      <span className="head-example"></span>
    </Badge>
  </div>
, mountNode);
````

````css
.am-badge {
  margin-right: 8px;
}
````
